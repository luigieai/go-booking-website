# go-booking-website

- Using GO 1.20.4
- Dependencies: [Chi Router](https://github.com/go-chi/chi/v5), [SCS Session management](https://github.com/alexedwards/scs/v2), [nosurf](https://github.com/justinas/nosurf) (already included at go.mod)
